# TCG Binder demo vault

This vault ships with a small sample: a collection, a 60-card deck and a few card notes under `TCG Binder/`.

To try it:

1. Install the TCG Binder plugin files into `.obsidian/plugins/tcg-binder/` and enable it in Settings → Community plugins.
2. Click the layers icon in the ribbon to open the binder.
3. Explore: open the collection, open the deck, run "Add cards" from the command palette (needs internet for the card database).

Everything the plugin knows lives in the frontmatter of these notes — open any of them in source mode to see the data model.
