# TCG Binder 1.0.1

Security-focused patch release, addressing the Obsidian plugin review scan.

## Changed

- **All vault access is now scoped to the binder folder** (default `TCG Binder/`). The plugin no longer enumerates the vault (`getMarkdownFiles`) — it walks only its own folder tree, so file paths outside it are never seen. Privacy by construction, and faster on large vaults.

## Breaking

- Manual card notes (and any plugin-managed note) must now live **inside the binder folder** to be recognized. If you kept card notes elsewhere, move them into the folder configured in Settings → TCG Binder.

## Notes for reviewers

- **Clipboard access** is write-only and user-initiated: the "Export deck" command copies a decklist in the Pokémon TCG Live format for pasting into the official client. The plugin never reads the clipboard.
