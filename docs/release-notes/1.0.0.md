# TCG Binder 1.0.0

First public release. TCG Binder manages your Pokémon TCG collections, decks and cards as plain Markdown notes inside your vault — your data stays readable, syncable and linkable, with or without the plugin.

## Highlights

- 📇 **Collections in Markdown** — every collection, deck and card is a note with frontmatter; the storage format *is* the export
- ⚡ **Bulk entry** — paste a TCG Live list (`4 Pikachu ex SVI 45`) or a CSV from ManaBox/Collectr/TCG Collector and import hundreds of cards at once
- 🃏 **Deck building with live validation** — 60-card rule, copy limits (basic energy exempt) and per-card format legality, plus TCG Live import/export
- 🎯 **Set tracking** — generate a full checklist for any set (secrets included) and watch the completion bar fill
- 💰 **Free price tooling** — market prices, cost-to-completion per set and a local portfolio value history chart

## Added

### Collections
- Card search by name or set + number (`SVI 45`), with card images in the results
- Rapid entry loop: search → variant/condition/quantity → keep searching
- Variant lines per card (normal, holo, reverse holo, promo) with inline editing; duplicate lines merge automatically
- Condition tracking (NM–DMG) with color-coded badges
- Playset indicator when you own 4+ copies of a card
- Set collections: full checklist of any set at quantity 0, with per-set completion % and cost to completion
- Wishlists (excluded from deck availability and portfolio value)

### Decks
- Import decklists in the Pokémon TCG Live text format; export back to the clipboard
- Live validation: deck size, copy limits with basic-energy exemption, format legality (Standard/Expanded/Unlimited)
- Pokémon/Trainer/Energy grouping with card thumbnails
- "Missing from collection" report with the cost to complete the deck

### Import & data
- CSV import with header detection for ManaBox, Collectr and TCG Collector exports
- Card database: TCGdex by default (free, no key); pokemontcg.io supported with an API key
- Offline-friendly: set catalog and set contents cached on disk with stale fallback
- Manual cards: any note with the right frontmatter is a valid card, even if missing from the database

### Pricing
- Market prices (TCGplayer, USD) stamped on card notes with the snapshot date
- "Update prices" command with portfolio value snapshots and a history chart

### Interface
- Binder dashboard with stats, collections, decks and portfolio chart
- Follows your Obsidian theme (light/dark) and respects reduced motion
- English and Brazilian Portuguese localization

## Install

Not yet in the community store. Either:

- **BRAT**: add `bgarciamoura/obsidian-tcg-binder` as a beta plugin, or
- **Manual**: download `main.js`, `manifest.json` and `styles.css` below into `<vault>/.obsidian/plugins/tcg-binder/` and enable TCG Binder in Settings → Community plugins.

Requires Obsidian 1.8.7+. Works on desktop and mobile.
